# CodeAlpha EDA Task - Student Marks Analysis

A simple Exploratory Data Analysis project to analyze student marks and study hours using Python, Pandas, and Matplotlib.

📊 Project Overview
This project performs basic EDA on a sample student dataset to calculate statistics and visualize marks distribution.

🔧 Tech Stack
- *Python 3.10*
- *Pandas*- Data analysis
- *Matplotlib* - Data visualization

 📁 Files
| File | Description |
| --- | --- |
| `eda_task.py` | Main Python script with EDA functions |
| `marks_analysis.png` | Bar chart visualization of student marks |
| `requirements.txt` | Required Python libraries |
